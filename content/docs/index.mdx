## Project goals

- Create a project framework that reduces the time needed to build standard website features.
- Create a design system with documentation that guides the team in decision making.
- Create a Figma framework that simplifies the creation and handoff of design.
- Develop a set of standard site features that can launch with all new sites.
- Develop a framework for build custom site features that fall outside of a base build.
- Use a more agile approach to site builds, allowing content, design and development to work in parallel.
- Create a WordPress pattern library that houses a collection of patterns our team can use to rapidly prototype and build new sites.
- Create WordPress plugin that houses custom functionality our team uses in all sites.
- Create a test library for testing theme and plugin changes.
