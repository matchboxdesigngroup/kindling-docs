## Projects start with matchbox-wp

The <a href="https://github.com/matchboxdesigngroup/matchbox-wp" target="_blank" rel="noopener noreferrer">
	matchbox-wp
</a> repo is a public GitHub template used to start our WordPress projects.

### Key features

- The Kindling theme.
- A plugin stub for custom, project-specific features.
- A root-level build system.
- Composer for dependency management.

## Core Features plugin

The repo contains a mu-plugin stub for custom functionality built specifically for that website. If your project doesn’t need the plugin, please delete it.

### Configuration

The plugin’s text domain and package name will not change from project to project. However, we set the plugin’s name to “<em>[Client Name] – Core Features</em>” so that it can be easily identified as belonging to the site rather than a service like SpinupWP.

> By keeping the functionality separate from the theme, we don’t have to rebuild core features when updating the theme in future redesigns.
